Software Configuration

To begin the software configuration, connect the PAM as described in the connection section. Once the ROV is connected to the camera properly, power on the laptop and turn on the panel. The light on the PAM should blink twice as it executes the bootloader, and then emit a single blink every few seconds if it is receiving power. When the PAM begins communicating, it will emit a series of blinks every few seconds. Before we begin configuring the software, we want to test that the PAM is connected correctly.

  1. Open cockpit.
  2. Find the camera instrument. If it does not appear, click the show all instruments button in the bottom pane. The default position is at the top left of the cockpit window.
  3. Click the switch button on the camera instrument. This should switch the video feed in the main window to the zoom camera.

Currently the buttons on the camera instrument will not work. If the video feed test was successful, we can now configure the software to enable them.

  1. Close VideoRay Cockpit.
  2. Navigate to the cockpit installation directory. The default directory is C:\Program Files (x86)\VideoRay\vrCockpit.
  3. Open vrDeviceConfiguration.exe.
  4. Click the ? button at the bottom of the utility to enumerate the PAM.
  5. If your PAM is connected correctly, you should see a listing in the left window of vrDeviceConfig. Click it.
  6. Click the ID button at the bottom of the utility to fill in the ID field.
  7. In the Device RS485 dropdown menu, select "Yes."
  8. Leave all other options as they are and click the floppy disk icon at the bottom to save your settings.

This configuration should enable the software functions of VideoRay Cockpit. Restart VideoRay Cockpit and test the camera's zoom function. You should be able to zoom in and out with the camera using the up and down arrows.
